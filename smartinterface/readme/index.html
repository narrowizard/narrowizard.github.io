SMARTINTERFACE

Smart Interface使用说明

简介

smart interface是一个基于tinygo和tinyts的接口文档管理系统。

特性

  • 项目私有化
  • 项目订阅
  • 接口日志
  • 接口标签
  • 数据统计
  • 多语言支持(json,typescript,java,golang)
  • 接口搜索(参数、标签搜索)
  • api支持

开始使用

  1. 打开smart interface,地址: https://smartinterface.narro.me/login
  2. 注册帐号(注意需要验证邮箱)或者使用第三方帐号登录。

  3. 登录系统后新建一个项目,选择是否公开,填写项目名称以及项目描述。
